Branching and Merging using GitHub

Hello Everyone,

This is my first experience with Github and I think I should share my knowledge on creating and merging of new branch.

Creating branches :
1)To create a new branch:
Command: git branch branchName

2)To switch head pointer to new branch:
Command: git checkout branchName

Above two steps can be combined using a single command:
Command: git checkout -b branchName

For merging files in master :
1)Head pointing to master:
Command: git checkout master

2)Now merge the files from branchName to master:
Command: git merge branchname

To delete the branch:
Command: git branch -d branchname

To check all the branches available and currently working branch:
Command: git branch

Note: * prefix to a branch name will indicate currently working branch.

Difference between git push and git push origin branchName
Command: git push
Above pushes change to master
Command: git push origin branchName
Above pushes newly created branch or changes made to that branch.

For collaborators to work on new branch:
Command: git pull
Command: git checkout branchname
