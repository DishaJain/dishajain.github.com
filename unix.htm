My first experience with UNIX


Hi all..
I am a beginner in UNIX and thought of exploring it. I tried out many basic commands and These were really helpful to me as a beginner in this language and I hope it would be of worth for many people like me.

  • This example works similarly like a search option we have in windows.
    Command: grep -Rs hi /home/appdev/temp/*
    Output:
    /home/appdev/temp/disha1/disha2.txt:hi
    /home/appdev/temp/disha1/disha1.txt:hi day1

    Above command will search for keyword "hi" in the  directory path "/home/appdev/temp/*"
    the R option makes it recursive, and the s will suppress any error messages about non-existent files or directories.
  • How to return back to home directory?
    Command: cd ~
  • How to remove the directory and all files in that directory?
    Command: rm -r directoryName
    Note:This command will exclusively remove the files residing in that directory along with directory.
  • How to remove all the files within a directory?
    Command:  rm *
    Note: This command will remove all the file residing in that directory.

    Hope this bolg works well for all beginners. Please pour in your suggestions.
